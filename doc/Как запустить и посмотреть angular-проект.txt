1. Установите nodejs на свой компьютер. Проверьте, установлены ли nodejs и npm (менеджер пакетов angular) в cmd
(или linux терминале) - nodejs --version и npm --version. При установке nodejs также устаналивается angular-cli - с ней
удобно генерить компоненты проекта из командной строки ангуляр. Проверьте, установлена ли она с помощью ng --version.
2. Зайдите в терминал в директории модуля miraculous-ui. Запустите команду ng serve - если терминал говорит "compiled
successfully" - запустите localhost:4200/ - angular по умолчанию запускается на 4200 порту.
