ПОРЯДОК РАБОТЫ С ДОСКОЙ TRELLO

Заведение новой карточки
1. Формулируем суть задачи
2. Определяем следующий индекс карточки (последнюю добавленную можно определить в карточке "Регистрация карточек")
3. Создаем новую карточку в колонке Product Backlog с именем "Индекс: Краткая суть задачи"
4. Добавляем в карточку "Регистрация карточек" в комментарии название созданной карточки
5. В карточке прописываем:
    - Название карточки должно быть таким "Индекс: Краткая суть задачи"
    - В Описание пишем подробную суть задачи
    - Добавить поэтапный чек-лист выполнения задачи в карточку
    - Добавляем файлы, относящиеся к задаче во Вложения
6. Ставим метку на карточку: зеленая - общие вопросы, желтая - фронтенд, синяя - бэкенд, красная - срочная.

Работа с карточками
1. Выбираете карточку с задачей в колонке "Sprint" (зеленая - общие вопросы, желтая - фронтенд,
синяя - бэкенд, красная - срочная)
2. Переносите ее в колонку "В работе"
3. Ставите себя в карточку в качестве исполнителя в разделе "Участники"
4. Ставите на карточку срок выполнения в разделе "Срок".
5. В среду пишем промежуточный отчет по задачам и проводим обсуждение
6. В пятницу сдаем pull request и проводим code review (если задача не выполнена/выполнена частино, то сдаем отчет)
7. В процессе выполнения задачи дюбые материалы, касаемые задачи (схемы, презентации, тексты и пр.)
обязательно прикрепляйте к карточке.
8. После того как задача была реализована (реализацией задачи признается отправленный pull request) переносите
карточку в колонку "Сделано за Sprint".

Среда
Сдаем промежуточный отчет, который состоит из ответов на следующие вопросы:
    - Что было сделано по взятой карточке?
    - Что предстоит еще сделать по взятой карточке?
    - Какие возникли сложности в процессе выполнения? Есть ли "затыки"?
    - Какие есть предложения по улучшению?

Пятница
1. Если задача выполнена, то создаете Pull request и переносите карточку в "Сделано за Sprint"
2. Если задача выполнена неполностью/невыполнена, то жду отчет, который состоит из ответов на следующие вопросы:
    - Что было сделано по взятой карточке?
    - Что предстоит еще сделать по взятой карточке?
    - Какие возникли сложности в процессе выполнения? Есть ли "затыки"?
    - Какие есть предложения по улучшению?

Общая информация
1. Любые вопросы задавайте либо в общий чат, либо в личку Алексею Щеглову.
2. Будем стараться общими усилия продвинуться вперед.
3. Если у Вас есть идеи по разделению задачи на несколько, но обязательно об этом сообщите - работа пойдет быстрее и проще.

Помните, что если Вы не можете оценить объем задачи, не укладываетесь в срок или столкнулись с любым недопонимаением
- сообщите об этом в обязательном порядке!!!
