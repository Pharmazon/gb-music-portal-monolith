КАК ЗАПУСТИТЬ ФРОНТ НА ANGULAR

1. Установите Node.js на свой компьютер.
2. Проверьте, корректно ли установился Node.js командой в консоли (node --version)
3. Проверьте, корректно ли установился npm командой в консоли (npm --version)
4. Зайдите в директорию проекта "cd /frontend/miraculous-ui/"
5. Выполните в консоли команду npm install и должитель установки бибилиотек.
6. Выполните в консоли команду npm start, должно вернуться сообщение "Compiled successfully"
7. Откройте в браузере http://localhost:4200
